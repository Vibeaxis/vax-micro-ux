VAX Micro UX

Micro-interaction helpers for WordPress. Small, fast, receipt-first UX tweaks from VibeAxis.

GitHub Repo VibeAxis Hugging Face

What’s inside

  • Sticky-header friendly anchors (no more buried headings).
  • Copy-link icons on H2–H4 (keyboard accessible).
  • Smooth scroll that respects prefers-reduced-motion.
  • Optional double-submit guard + tiny progress cues.

Install

Drop the plugin into /wp-content/plugins/ and activate. Settings live under Settings → VAX Micro UX.
